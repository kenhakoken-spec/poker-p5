# 手動テスト チェックリスト

## 前提

- ブラウザで http://localhost:3000 を開く
- ビューポートを Pixel 8 Pro 相当 (412×915) に設定して実施推奨

## 1. TOP 画面

- [ ] 勝率・記録数・記録開始ボタン・「タップでメニュー」がすべて見える
- [ ] 縦スクロールが発生しない
- [ ] タップでメニュー展開 → 履歴・設定・解析が表示される
- [ ] 記録開始をタップすると /record に遷移する

## 2. 記録開始

- [ ] 「ハンド記録開始」の文字がジャキーンで表示される
- [ ] 「開始」ボタンでヒーロー選択画面に遷移する

## 3. ヒーロー選択

- [ ] 6ポジションが表示される
- [ ] 1つ選ぶとハンド（オプション）の1デッキが表示される
- [ ] 1枚目選択 → 2枚目選択で2枚確定する
- [ ] 「やり直す」でクリアできる
- [ ] 「開始」で Who is Open? に遷移する
- [ ] 縦スクロールが発生しない

## 4. Who is Open?

- [ ] UTG, MP, CO, BTN, SB, BB, No open が選択できる
- [ ] いずれかを選ぶとアクション選択画面に遷移する
- [ ] Pot とストリート表示がある

## 5. プリフロ アクション

- [ ] 選択したオープナー（または BB）のアクション（fold / bet / call / raise / all-in）が選べる
- [ ] アクション後、次のポジション選択または勝者選択に遷移する
- [ ] 全員フォールドで「勝者は？」に遷移する
- [ ] ラウンドが閉じるとフロップ3枚選択に遷移する

## 6. フロップ・ターン・リバー ボード選択

- [ ] フロップで3枚、ターンで1枚、リバーで1枚選べる
- [ ] 使用済みカード（ホール＋既出ボード）は除外されたデッキから選ぶ
- [ ] 「確定」で次のストリートのアクションに進む
- [ ] 縦スクロールが発生しない

## 7. ポストフロップ アクション

- [ ] ストリート順（SB→BB→...→BTN）でポジション選択・アクションができる
- [ ] 全員チェックで次ストリートへ
- [ ] ベット-コールで次ストリートへ
- [ ] 1人残りで「勝者は？」に遷移する
- [ ] リバー完了で「勝者は？」に遷移する

## 8. 勝者・ショーダウン・結果

- [ ] 勝者を1人選べる
- [ ] ショーダウン（オプション）で「次へ」で結果画面へ
- [ ] 勝利/敗北で TOP に戻る
- [ ] 履歴に1ハンド追加されている

## 9. 記録画面の P5 デザイン

- [ ] Pot が大きく表示される
- [ ] ストリートが斜めラベルで表示される
- [ ] 直近アクションが表示される（ActionHistory）
- [ ] 縦スクロールが発生しない

## 代表シナリオ（最低1本は実施）

1. **最短**: Who is Open = UTG → UTG fold → 続けて全員フォールド → 勝者 BB → 次へ → 勝利 → TOP
2. **フロップまで**: Who is Open = CO → CO 3x → BTN call → SB fold → BB call → フロップ3枚選択 → 全員チェック → ターン1枚 → 全員チェック → リバー1枚 → 全員チェック → 勝者選択 → 次へ → 結果
3. **ヒーローBTN**: ヒーロー BTN、Who is Open = UTG → UTG 3x → MP fold → CO fold → BTN call → SB fold → BB call → フロップ3枚 → … → 結果

以上を実施し、問題なければ E2E 合格とする。
